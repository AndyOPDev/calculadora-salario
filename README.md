# Calculadora de Sueldo Neto · España 2025

Calculadora interactiva para estimar el **sueldo neto mensual y anual** a partir del salario bruto, aplicando la normativa IRPF 2025 (estatal + Murcia) y cotizaciones a la Seguridad Social.

## 🚀 Demo

[https://andyopdev.github.io/calculadora-salario/](https://andyopdev.github.io/calculadora-salario/)

## ✨ Características

- Cálculo de IRPF estatal + autonómico (Murcia 2025)
- Cotización a la Seguridad Social (~6,35%)
- Reducción por rendimientos del trabajo (art. 20)
- Mínimo personal (5.550 €)
- Soporte para plus de disponibilidad / guardias
- Ayudas exentas de IRPF (suministros, comunicaciones)
- Selección entre 12 o 14 pagas
- Visualización de tramos IRPF aplicados
- Gráfico donut: distribución Neto / IRPF / SS
- Diseño responsive y modo oscuro

## 📊 Datos incluidos

| Concepto | Valor |

## 🧮 Uso

1. Introduce tu **salario bruto anual**
2. (Opcional) Añade plus de disponibilidad mensual
3. (Opcional) Añade ayudas exentas mensuales
4. Selecciona **12 o 14 pagas**
5. Visualiza resultados: neto por paga, neto anual, desglose, tramos IRPF y gráfico

## 📝 Notas

- Los cálculos son **estimativos** y pueden variar según circunstancias personales
- Basado en normativa IRPF **2025** para la **Región de Murcia**
- La reducción por rendimientos del trabajo aplica automáticamente
